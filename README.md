# AFK

### What is this?
This is the source code of a minecraft mod written in Java.

### Description
A simple AFK mod to make multiplayer sleeping easier.

### Commands
| - means <b>or</b></br>
[] - means <b>required</b></br>
<> - means <b>optional</b>

<details>
<summary>/afk</summary>
<b>Description: Toggles afk mode</b>
<br>
<b>Permission(s):</b> none
</details>
